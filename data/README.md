# Data

Holds task data used for evaluation, or instructions to obtain it.

Planned tasks (from Yao et al., 2023):
- **Game of 24** — 1,362 puzzles from 4nums.com; the paper evaluates on indices 901–1000.
- **Creative Writing** — 100 random sentence sets used as paragraph-ending constraints.
- **Mini Crosswords** — 5x5 crosswords from GooBix.

For each task, place raw files under `data/<task>/` or document the download steps here.
